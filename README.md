# hasm-vscode
Human Assembler for Visual Studio Code

Enlace al proyecto principal del lenguaje: https://github.com/pekesoft/hasm

## Tipos de ficheros soportados

### .hasm

Fichero de código general en hasm.

### .mhasm

Fichero de macros.

### .hinc

Fichero de includes.

### .hlib

Librería de funciones.
